Telegram-система мониторинга валютных курсов

Telegram-бот для мониторинга валютных курсов с хранением истории,
уведомлениями об изменениях и гибкими пользовательскими настройками.

Стек:
- Python 3.13
- aiogram 3
- PostgreSQL
- HTTPX / BeautifulSoup
- APScheduler

Функциональность:
- выбор валют для мониторинга
- получение текущих курсов
- автоматический парсинг данных по расписанию
- хранение истории изменений
- уведомления при изменении курса (порог задаётся при подписке)
- управление настройками мониторинга

Архитектура:
- Telegram-бот (aiogram) — пользовательский интерфейс
- Парсер — отдельный сервис сбора данных
- PostgreSQL — хранение пользователей, подписок и истории
- Планировщик — периодический сбор и проверка изменений

Примеры использования:
- мониторинг валютных курсов
- уведомления для трейдеров
- финансовые боты
- интеграция в бизнес-системы

Проект может быть адаптирован под:
- акции
- криптовалюты
- товары
- индексы


Коммерческое использование:

Проект может быть использован как основа для:
- финансовых Telegram-ботов
- систем мониторинга данных
- уведомлений для бизнеса
- кастомных решений под конкретные требования

Возможна адаптация и доработка под задачи заказчика.


Запуск проекта:

1. Склонировать репозиторий
2. Создать файл .env на основе .env.example
3. Установить зависимости:
   pip install -r requirements.txt
4. Запустить бота:
   python main.py
